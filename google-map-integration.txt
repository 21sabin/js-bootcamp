 **********Integrating Google Map *************
1.Generate a google dev project at http://console.developers.google.com
2.Enable Google Maps support inside the project.
 foreg Enable Maps Javascript API
3.Generate an Api key
4.Add the google maps script tag to our html file. 


stpes:
1. After Enabling the service click on the hambergure icon and go to the credentials.
2.Click on the create credential and create API key;
3.inside your html file create a script tag and include google link.
e.g <script src="https://maps.googleapis.com/maps/api/js?key=<api_key>"></script>
4.To test google api has been integrated or not go to the developer section and inside console type google and hit enter.
 If you get the maps object then it is good to go.
